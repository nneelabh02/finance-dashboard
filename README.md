# Finance Dashboard 💰

A modern finance dashboard built with React, TailwindCSS, and Zustand.

## Features

* 📊 Transactions tracking
* 🌙 Dark/Light mode (persistent)
* 📱 Responsive design
* 🔍 Search & Sort
* 📤 Export (CSV / JSON)
* ⚡ Smooth animations

## Tech Stack

* React + Vite
* TailwindCSS
* Zustand
* Framer Motion
* json-server (mock API)

## Run Locally

```bash
npm install
npm run dev:full
```

This runs both the frontend app and the mock API simultaneously using `concurrently`.

If you prefer separate processes:

```bash
npm run dev
npm run server
```
