# Finance Dashboard 💰

A modern finance dashboard built with React, TailwindCSS, and Zustand.

## Features

* 📊 Transactions tracking
* 🌙 Dark/Light mode (persistent)
* 📱 Responsive design
* 🔍 Search & Sort
* 📤 Export (CSV / JSON)
* ⚡ Smooth animations

## Tech Stack

* React + Vite
* TailwindCSS
* Zustand
* Framer Motion
* json-server (mock API)

## Run Locally

```bash
npm install
npm run dev
```

Start mock API:

```bash
npx json-server --watch db.json --port 3000
```
